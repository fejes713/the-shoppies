### Project
Live demo of the project can be accessed on this [link](https://zen-einstein-689da3.netlify.app/)

### Development Setup
- Clone the repo using your favorite approach
- Navigate to your terminal and do the following
- `cd ~/the-shoppies`
- `npm install`
- `npm start`
